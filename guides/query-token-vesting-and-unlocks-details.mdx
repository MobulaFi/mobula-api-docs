### Walkthrough

  <Step title="Execute the API call">
    You can call Mobula API directly from your terminal using cURL or any HTTP client.

    **Using cURL**:
    ```bash
    curl --request GET \
    --url 'https://api.mobula.io/api/1/metadata?asset=avalanche'
    ```

    **Using Axios**:
    ```javascript
    import axios from "axios";
    axios.get('https://api.mobula.io/api/1/metadata?asset=avalanche')
    .then(function (response) {
      console.log(response);
    })
    .catch(function (error) {
      console.log(error);
    });
    ```

  </Step>
