<Tip> This endpoint is only available to Growth and Enterprise plans. </Tip>

### Endpoint details

- URL: wss://portfolio-api-wss-fgpupeioaa-uc.a.run.app
- Message (string):

```json
{
  "type": "wallet",
  "authorization": "YOUR-API-KEY",
  "payload": {
    "wallet": "0x6557D92D629E0bD8003b8E12c83835ff3CDB9D6B",
    "interval": 15
  }
}
```

Parameters:

- `interval`: interval in seconds between updates (min 5s, default 30s)
- `wallet`: wallet address to track
- `wallets`: array of wallet addresses to track (comma separated)
- `type`: 'history' or 'portfolio' (default 'portfolio')

### Data model

Data reponse is the same as [wallet-portfolio](/rest-api-reference/endpoint/wallet-portfolio)
